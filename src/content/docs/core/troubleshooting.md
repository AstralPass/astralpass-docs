title: Troubleshooting

---

This page covers the most common user-side problems in AstralPass.

## My wallet will not link

Try these basics first:

- make sure the wallet you expect is actually selected
- retry the ownership approval step
- if you use a hardware wallet, confirm the request on the device
- refresh and try again

## My roles did not update

Possible reasons include:

- you still need to complete another onboarding step
- the DAC requires manual approval first
- the linked wallet does not satisfy the relevant rule
- role sync has not run yet

## I linked the wrong wallet

If you linked the wrong wallet, remove it and link the correct one.
If you use multiple wallets, make sure the wallet with the relevant DAC or Star Atlas state is actually connected to AstralPass.

## I use a Ledger or another hardware wallet

Hardware wallets can behave differently from hot wallets.
That is expected.

Extra prompts, device confirmations, or different signing behavior do not automatically mean something is broken.

## I signed in, but I still cannot access what I expected

In most cases, one of these is still unresolved:

- the wrong wallet is linked
- Discord-side approval is still incomplete
- you have not joined the DAC on-chain with the right wallet
- AstralPass has not re-evaluated your state yet

## Still stuck?

If you are still stuck after checking the basics, contact the DAC team or support path that sent you to AstralPass in the first place.
